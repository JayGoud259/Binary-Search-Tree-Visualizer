# Binary-Search-Tree-Visualizer
A Binary Search Tree Visualizer that demonstrates BST operations like insertion, deletion, and height calculation to help students understand tree data structures.
Binary Search Tree Visualizer

Project Description

Understanding tree data structures can be difficult for students because tree operations occur internally and are not always easy to imagine. A visual system that demonstrates these operations step-by-step can help students understand the working of trees more clearly.

In this project, we develop a Binary Search Tree (BST) Visualizer that demonstrates important tree operations such as insertion of nodes, deletion of nodes, and height calculation of the tree. The system allows users to enter values and see how they are placed in the Binary Search Tree according to BST rules.

When a node is inserted, the system places it in the correct position in the tree. When a node is deleted, the tree structure updates accordingly. The system can also calculate the height of the tree, which represents the longest path from the root node to the deepest leaf node.

This project helps students understand how Binary Search Trees work and how operations like insert, delete, and height calculation affect the structure of the tree.

Data Structures Used

Binary Search Tree (BST) – used to store and organize data in a hierarchical structure where each node follows the BST rule (left child contains smaller values and right child contains larger values).

Tree Nodes / Linked Structure – used to represent each node with data and pointers to left and right child nodes.

Expected Features

Accept node values from the user
Insert nodes into the Binary Search Tree
Delete nodes from the Binary Search Tree
Calculate and display the height of the tree
Display the Binary Search Tree structure
Efficient management of data using BST properties
